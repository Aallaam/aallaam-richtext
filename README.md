# aallaam-richtext

## Installation
### npm
```
npm install aallaam-richtext
```

## Usage
```
<template>
    <div>
        <aallaam-richtext :withImage="true" />
    </div>
</template>
 
<script>
    import AallaamRichtext from 'aallaam-richtext'
 
    export default {
        components: {
            AallaamRichtext
        }
    }
</script>
```
